Banco de dados -> armazena um conjunto de dados de forma organizada

dados -> Dados são observações documentadas ou resultados da medição. A disponibilidade dos dados oferece oportunidades para a obtenção de informações. Os dados podem ser obtidos pela percepção através dos sentidos (por exemplo observação) ou pela execução de um processo de medição.

informações -> As informações são os dados devidamente tratados e analisados, produzindo conhecimento relevante. Ao contrário dos elementos brutos do tópico anterior, elas têm significados práticos e podem ser utilizadas para reforçar o processo de tomada de decisão.
