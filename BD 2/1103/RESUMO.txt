MER - modelo entidade - relacionamento

entidade -> objeto que do mundo real que guarda características
Atributos -> características das Entidades
Atributos chave ( Chave primária) -> Único em cada entidade
Chave estrangeira -> atributo chave da outra entidade que efetiva o relacionamento
(Forasteiro)
Relacionamento -> indica a relação entre duas entidades
Cardinalidade -> restrição que quantifica a relação entre duas entidades
Um para Um -> um elemento de uma entidade se relaciona com um elemento de outra entidade
Obs.: a chave estrangeira estará em um das duas entidades
Um para Muitos -> um elemento de uma entidade se relaciona com muitos elementos de outra entidade
Obs.: a chave estrangeira estará do lado muitos
Muitos para Muitos -> um elemento de uma entidade se relaciona com muitos elementos de outra entidade,
o elemento da primeira entidade se relaciona com o da segunda
Obs.: requer tabela extra para inserir as chaves estrangeiras
A leitura da cardinalidade sempre começa com Um elemento.
